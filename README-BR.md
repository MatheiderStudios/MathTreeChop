![MathTreeChop Banner](imagens/banner.png)

Um plugin de corte de árvores altamente otimizado, 99% inspirado no mod *Panda's Falling Tree*, trazendo uma experiência suave, natural e com animações personalizadas para servidores Minecraft.

---

## 🎬 Showcase

![MathTreeChop Showcase](imagens/cortando_arvore_grande.gif)

---

## ✨ Recursos

- 🌲 Corte árvores inteiras quebrando apenas um bloco
- ⚡ Animação suave personalizada (não é quebra instantânea)
- 🧠 Sistema otimizado para evitar lag no servidor
- 🔧 Compatível com múltiplos loaders
- 🎮 Experiência semelhante a mods, adaptada para multiplayer

---

## 🚀 Instalação

1. Baixe o arquivo `.jar` do plugin  
2. Coloque na pasta `/plugins` do seu servidor  
3. Reinicie o servidor  
4. Pronto!  

---

## ⚙️ Objetivo do Projeto

O objetivo do **MathTreeChop** é fornecer um sistema de corte de árvores completo que seja:

- leve  
- visualmente agradável  
- otimizado para servidores  

Sem necessidade de mods no cliente.

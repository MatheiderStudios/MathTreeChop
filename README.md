A highly optimized tree-chopping plugin, 99% inspired by the *Panda's Falling Tree* mod, bringing a smooth, natural experience with custom animations to Minecraft servers.

## ✨ Features

- 🌲 Chop entire trees by breaking a single block
- ⚡ Custom smooth animation (not just instant breaking)
- 🧠 Optimized system to prevent server lag
- 🔧 Compatible with multiple server loaders
- 🎮 Mod-like experience adapted for multiplayer servers

## 🚀 Installation

1. Download the plugin `.jar`
2. Place it inside your server `/plugins` folder
3. Restart the server
4. Done!

---

## ⚙️ Project Goal

The goal of **MathTreeChop** is to provide a full tree-cutting mechanic that is:

- lightweight  
- visually appealing  
- server-friendly  

Without requiring any client-side mods.
